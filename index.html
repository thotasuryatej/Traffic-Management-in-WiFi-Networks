Modeling and Enhancing IoT Traffic Management in WiFi Networks

Simulator: NS-3

Objectives:

  • Setting up Environment
  • Normal Routing Performance
  • Equal Bandwidth Distribution
  • Reducing Latency
  • Maintain Stable Performance

Node Creation Input

nCsma: Number of extra CSMA nodes/devices to be created.

nWifi: Number of Wi-Fi STA devices to be created.

verbose: Enables logging for echo applications if set to true.

tracing: Enables pcap tracing if set to true, which logs packet-level data in PCAP format.

asciitracing: Enables ASCII tracing if set to true, which logs data in a human-readable ASCII format.

Title:

MANET Routing Protocol Performance Comparison

Purpose:

The code simulates and evaluates the performance of various routing protocols in Mobile Ad-Hoc Networks (MANETs) using the NS-3 simulator. It compares protocols like AODV, OLSR, DSDV, and DSR based on key metrics such as throughput, packet delivery ratio, and delay.
